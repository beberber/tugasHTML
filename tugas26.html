mengenal semantic elements di HTML5

  • artikel

    • ketika kita ingin membuat layout untuk mennampilkan block artikel yang ada di website kita. tag article bisa kita gunakan

    karena tag artikel memiliki type block artinya elemen ini memiliki lebar sesuai lebar browser. jika kita tidak menentukan lebar.

  • aside

    • elemet ini bisa kita gunakan untuk melengkapi layout utama. misal kita memiliki halaman blog maka kita bisa menambahkan

    karena aside memiliki type/sifat block maka lebar sesuai page jika kita tidak menentukan lebarnya.
